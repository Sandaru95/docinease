Introduction to our tool

What we are trying to achieve is to give the user the full potential to use OCR (Optical Character Recognition) technology as they wanted. With OCR the user will be able to convert their handwritten documents into typed pdf files. So we are continuously working to improve the product

How To Convert Handwritten Text To PDF files?

Just tap on the upload button and select the handwritten document image. After doing that tap on the proceeding button. Then in a few seconds, your document will be converted into a PDF file.

Manipulation of Saved PDF files

So after uploading the handwritten document and saving it as a PDF, You'll have the options to view info, print and download the generated PDF file.

How to Print Generated PDF file

All you have to do is tap on the print button of the relavant PDF file and the hit the (Ctrl + P)
